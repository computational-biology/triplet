A Brief Intro
In this work, we report a software tool, called BPNet, that accepts a mm- CIF or PDB file and computes the base-pair/contact network components using graph formalism. The software can run on linux platform in both serial and parallel modes. It generates several information in suitable file formats for visualization of the networks.
